Make sure to review the results from user tests right away. Pretend like you're really hungry at Texas Roadhouse and the only way you're allowed to eat is if you complete the user testing debrief (Texas Roadhouse is SO GOOD!!! I LOVE TEXAS ROADHOUSE!!!).

"Obvious"

The things that you need to change about your interface will be obvious. If they're not obvious, then you really need to think critically about what to change and make sure that your changes are very small. A minor change can have a major impact. Don't break the parts that already work.

So what?

Don't listen to everything a user has to say. Don't add a bunch of things to appease a few users. Take things away. Addition by subtraction. Be intentional and make small changes.
